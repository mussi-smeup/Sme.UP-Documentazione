# OBIETTIVO
Gestire le operazioni base sull'oggetto Q5 Fonte.
Ogni oggetto ha associate delle fonti standard e altre possono essere aggiunte tramite la compilazione del file di script SCP_QRYPER.

# FUNZIONI E METODI

## DAT - Ritorno dati
Viene ritornta in £IQ5OU la DS £IQ5D contenente le proprietà della fonte.

>   £IQ5DCD Codice
   £IQ5DDE Descrizione
   £IQ5DPG B£IQR_xx
   £IQ5DPF Prefisso query


## LIS - Lista fonti
Viene ritornata la sequenza delle fonti registrate per un oggetto.
Dalla fonte vengono fornite le query disponibili per l'oggetto.

## LIS.QRY - Lista query
Viene ritornata la sequenza delle query disponibili per la fonte.
