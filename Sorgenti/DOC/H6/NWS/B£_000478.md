E' disponibile la nuova routine £G64 che consente di gestire le funzioni su una Coda Dati esistente.

Le funzioni implementate sono le seguenti : 

           -  SND  :  scrittura coda
           -  RCV  :  lettura coda
           -  CLR  :  pulizia coda

La £G64 gestisce anche le Code Dati con chiave.
