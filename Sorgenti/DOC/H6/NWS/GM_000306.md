Sono stati introdotti i seguenti campi alfanumerici  : 
   -  N° 53  :   Numero Ordine di Produzione 1
           ( presente solo se l'ordine 1 è di produzione )

   -  N° 54  :   Numero Ordine di Produzione 2
           ( presente solo se l'ordine 2 è di produzione )

   -  N° 55  :   Numero Ordine Esterno 1
           ( ordine + riga )  : 
           ( presente solo se l'ordine 1 è di ciclo esterno )

   -  N° 56  :   Numero Ordine Esterno 2
           ( ordine + riga )  : 
           ( presente solo se l'ordine 2 è di ciclo esterno )

 In questo modo è possibile , negli schemi , agganciare gli OAV di questi Oggetti .

Ricordo che , se l'ordine è di ciclo esterno , il tipo è presente nei campi  " Tipo Ordine 1/2 " ,
e quindi può essere usato per individuare l'oggetto.
